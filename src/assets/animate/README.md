# Landing page images

Drop image files here (`.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`) and they’ll appear on the landing page in this order:

1. **Hero** – main visual (replaces default speaker image)
2. **Ecosystem – Freelancers** – card header image
3. **Ecosystem – Production Houses** – card header image
4. **Ecosystem – Vendors** – card header image
5. **Features – Calendar** – “Real-time Centralized Calendar” visual
6. **Features – Workflow** – “Structured Hiring Workflows” visual
7. **How it works – Search** – step 1 visual
8. **How it works – Book** – step 2 visual
9. **How it works – Manage** – step 3 visual
10+ **CTA** – extra thumbnails above the call-to-action (optional)

If you add fewer than 9 images, the rest of the sections keep their default look (icons/mockups).
